# Studio Split Master Plan

Current State:
- Design Studio and Marketing Studio are mixed together.
- Asset creation, campaign creation, publishing and queue management exist in one module.

Target State:

Design Studio
- Brand Studio
- Asset Studio
- Creative Studio
- Document Studio
- Template Studio

Marketing Studio
- Campaign Builder
- Calendar
- Publishing
- Social Channels
- Analytics
- AI Optimization

Shared Layer
- Asset Library
- Generation Jobs
- AI Engines
- Tenant Security
- Organization Security

Rule:
Design Studio creates assets.
Marketing Studio consumes assets.
